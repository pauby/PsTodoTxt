# Get-TodoTxtTodaysDate

## SYNOPSIS
Gets todays date in todo.txt format.

## SYNTAX

```
Get-TodoTxtTodaysDate [<CommonParameters>]
```

## DESCRIPTION
gets todays date in the correct todo.txt format.

## EXAMPLES

### EXAMPLE 1
```
Get-TodoTodaysDate
```

Outputs todays date.
